# RaGLanceDbPoC

PoC implementation of AI/ML Prediction using RAG (Retrieval Augmented Generation) based on LanceDB

## Get started

1. Install dependencies

```shell
pip install -r requirements.txt
```

2. Run RAG demo

```shell
python main.py
```
